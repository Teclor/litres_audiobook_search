# litres_audiobook_search
Downloads and stores data about audiobooks, provides restful service for searching them by name
<hr>
Usage examples: <br>
https://{SERVER_NAME}/getAudiobookByName/?bookName={URL_ENCODED_STRING} <br>
/getAudiobookByName/?bookName=Just+so+Stories+%2F+Сказки <br>
/getAudiobookByName/?bookName=Three+Musketeers+%2F+%D0%A2%D1%80%D0%B8+%D0%BC%D1%83%D1%88%D0%BA%D0%B5%D1%82%D0%B5%D1%80%D0%B0
